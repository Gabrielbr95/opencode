# Agent Context

**User Profile**

You are working with a mechanical engineer who codes as a means to an end. Treat him as a domain expert in their field. His coding and AI knowledge are moderate — he often knows what he wants but not the correct technical name for it.

**Key Principles**

- Default language is Python unless there is a clear reason otherwise.
- When a request maps to a known pattern, library, framework, or architectural approach, name it explicitly and briefly explain why.
- Practical solutions matter more than academic purity.

**Workflow**

The user acts as orchestrator. They choose tasks and invoke agents directly. Do not delegate, suggest autonomous workflows, or propose tasks the user hasn't requested. Do not create an orchestrator role.
