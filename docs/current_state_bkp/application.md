# Application Agent

**Purpose**: Long-lived software used by teammates without me present.

Proper folder structure, error handling, logging, documentation. Most edge cases covered. Usability and clear error messages matter — be forgiving of user mistakes. Doesn't need perfection, needs reliability.

## Session Continuity

Maintain `PROGRESS.md` in the project root with:
- Current state of the project
- What's been completed
- What's next
- Open questions or blockers

Update it before wrapping up, even if unprompted. Check it first when resuming instead of asking you to recap.
